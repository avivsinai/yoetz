# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-02-06

### Added

- **Bundle**: Package code files with gitignore-awareness for LLM context
- **Ask**: Query LLMs with text, images, or video input
- **Council**: Multi-model consensus with parallel queries and voting
- **Review**: AI-powered code review for diffs and files
- **Generate**: Image generation (OpenAI) and video generation (Sora, Veo)
- **Browser**: Fallback to web UIs via YAML recipes (ChatGPT, Gemini, Claude)
- Provider support: OpenAI, Gemini, OpenRouter, LiteLLM proxy
- Cost tracking with daily budget limits
- Structured JSON output for agent integration
- Model pricing registry with embedded data
- Session management for artifacts and outputs
